# ဦးပုည ထီပေါက်ကိန်း (U Ponnya Lottery Fortune Calculator) 🎰

မြန်မာ့ရိုးရာ ဂြိုဟ်သက်ဗေဒင်ပညာရပ်နှင့် ဆရာကြီးဦးပုည၏ နည်းစနစ်ကို အခြေခံ၍ တွက်ချက်ပေးသော Digital ထီပေါက်ကိန်း Calculator ဖြစ်ပါသည်။

## 📌 နိဒါန်း (Introduction)
ဤ Project သည် ရှေးဟောင်းမြန်မာ့ဗေဒင်ကျမ်းလာ ဦးပုည ထီပေါက်ကိန်း တွက်ချက်ပုံကို ခေတ်မီ Web Technology (HTML, CSS, JavaScript) အသုံးပြု၍ ဖန်တီးထားခြင်း ဖြစ်ပါသည်။ အသုံးပြုသူ၏ မွေးသက္ကရာဇ်၊ မွေးနေ့နှင့် လက်ရှိရောက်သက်တို့ကို အခြေခံ၍ ထီကံဇာတာကို ဟောကိန်းထုတ်ပေးပါသည်။

## ✨ အဓိက လုပ်ဆောင်ချက်များ (Features)
* Traditional Logic: ရှေးမူမပျက် ဂြိုဟ်သက်တမ်းများဖြင့် တိကျစွာ တွက်ချက်ပေးခြင်း။
* Modern UI: ရွှေရောင်နှင့် အနီရင့်ရောင်ကို အခြေခံထားသော ဆွဲဆောင်မှုရှိသည့် Design။
* Responsive Design: ဖုန်း၊ တက်ဘလက် နှင့် ကွန်ပျူတာ အားလုံးတွင် ကြည့်ကောင်းအောင် ပြုလုပ်ထားခြင်း။
* Fast & Lightweight: Single code file ဖြစ်၍ မြန်ဆန်ပေါ့ပါးခြင်း။

## 🛠 အသုံးပြုထားသော နည်းပညာများ (Technologies Used)
* HTML5 - Page Structure အတွက်။
* CSS3 - Custom Styling နှင့် Animations အတွက်။
* JavaScript (ES6) - တွက်ချက်မှု Logic များအတွက်။

## 📖 တွက်ချက်ပုံ နည်းစနစ် (Algorithm)
၁။ မွေးဖွားသော မြန်မာသက္ကရာဇ်ကို တည်ပါ။  
၂။ မိမိမွေးနေ့အလိုက် သတ်မှတ်ထားသော ဂြိုဟ်သက်ကို ပေါင်းပါ။  
၃။ ရရှိသော ရလဒ်ကို (၄၃) နှင့် စားပါ။  
၄။ စားကြွင်းကို လက်ရှိ ရောက်ရှိနေသော အသက် (ရောက်သက်) နှင့် ပေါင်းပါ။  
၅။ ၎င်းရလဒ်ကို (၇) နှင့် စား၍ ရရှိသော အကြွင်းပေါ်မူတည်ကာ ကံဇာတာကို ဆုံးဖြတ်ပါသည်။

> ဟောကိန်းများ:
> * (၅) ကြွင်းလျှင် - တစ်ဦးတည်းထိုး ပေါက်မည်။
> * (၀) ကြွင်းလျှင် - စပ်တူထိုး ပေါက်မည်။

## 🚀 အသုံးပြုပုံ (How to Run)
၁။ ဤ Repository ကို Clone လုပ်ပါ သို့မဟုတ် index.html ဖိုင်ကို Download ဆွဲပါ။  
၂။ မည်သည့် Web Browser တွင်မဆို index.html ကို Open လုပ်၍ အသုံးပြုနိုင်ပါသည်။
